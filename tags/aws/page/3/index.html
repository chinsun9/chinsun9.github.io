따라하기

  • 설정하고싶은 RDS 화면에 들어간다
  • 연결 & 보안 > 보안 > vpc 보안 그룹
  • 을 눌러서 현재 rds에 적용되어 있는 보안그룹 설정으로 들어간다
  • 인바운드 규칙 > 인바운드 규칙 편집으로 들어간다
  • 아마 이런식으로 위치무관으로 되어있을 수 있는데

  • 이 규칙을 이제 내 컴퓨터에서만 접근가능하도록 고쳐보자

    1. 소스 유형에 내 IP 를 선택한다
    1. 내 공인 IP 주소로 바뀌었는지 확인한다
