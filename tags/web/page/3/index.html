Smooth Page Transitions

  • Smooth Page Transitions With Javascript Tutorial를 보고 따라만들었다.

결과

  • 보러가기

후기

  • 영상 제목만 보고 단순히 css js를 이용한 트랜지션을 구현할 줄 알았는데,
  • node로 parcel, gsap, highway 를 사용해서 웹만드는 것을 배우게되었다.
  • parcel을 처음 써보았는데, 개발할 때도, 배포할 때에도 좋은 것 같다.
  • higyway와 gsap으로 애니메이션을 구현해 보았다.
  • parcel을 이용해 웹을 만들어 깃허브 페이지로 배포할 때,
  • parcel build index.html --out-dir docs --public-url ./
  • 위 처럼 옵션을 넣어줘야한다.
  • –out-dir docs 는 깃허브 페이지 설정할 때 디렉터리
  • –public-url ./ 는 이미지, css, js 경로가 망가지지 않도록한다.

github hexo blog web font 적용하기

  • 기본 한글 폰트가 너무 구려서 웹폰트라는 것을 적용시켜보았다.
  • http://deminoth.github.io/google-font-kor/ 한글 폰트
  • 나눔 고딕을 적용해보았다.

환경

  • 나는 hexo에서 icarus 테마를 사용하고 있다.

폰트 고르기

  • https://fonts.google.com/?subset=korean

theme 코드 변경

  • fontcdn 이라는 메소드가 있어서 그냥 활용했다.
  • icarus theme config 파일의 fontcdn이 google로 되어있어서 그대로 사용할 수 있었다.
  • 나눔고딕을 3순위로 했는데, 영어랑 한글이랑 부자연스러운 것 같아서 그냥 나눔고딕을 1순위로 올렸다.
  • 하는 김에 본문 폰트 사이즈도 키워주었다.

확인하기

  • 잘 갱신되었는지 확인한다.

배포하기 전 확인하기

  • hexo server에서는 최신 css 상태를 반영하는데,
  • 그런데 실제로 배포될 때는 css파일이 갱신이 안된다.
  • 그래서 public/css/default.css 파일을 삭제하고, 다시 hexo generate로 생성해야한다.
  • 그리고 배포한다.
