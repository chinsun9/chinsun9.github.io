vscode shell git bash

  • git bash 터미널을 vscode에 추가하고 싶을 때,
  • shellLauncher 확장도구를 사용해서 위와 같이 각 터미널 실행파일 경로를 지정해주었다

  • F1 > Shell Launcher: Launch > git bash 선택하여 사용한다.
