Open Port Check WEB (NodeJs)

  • https://www.yougetsignal.com/tools/open-ports/ 을 따라만들기, 클론 코딩

기능

  • 접속자의 공인 아이피 알려줌
  • 포트포워딩 테스트
  • common ports 목록
  • 모든 common port 테스트

express generator로 프로젝트 생성

IPv4 방식으로 보기

  • ip를 index view에 렌더링하여 보냄

is-port-reachable

  • is-port-reachable 패키지를 사용해 포트 오픈 여부를 알아온다
  • 당연한 말이지만 해당 포트를 리슨하는 서버가 존재해야 한다
  • host는 domain을 적어도 되고, ip로 적어도 된다.

check all common ports

  • req로부터 ip를 가져오고,
  • common_ports_list의 port를 가져와서 프로미스를 생성한다
  • await Promise.all(reqPromiseArray)로 요청을 병렬 처리하였다

후기

  • 처음에는 https://www.findip.kr/ 처럼 공인 아이피 주소를 알려주는 웹을 만들었는데,
  • 너무 싱겁게 끝나서, 포트포워딩 체크하는 기능을 추가해보았다.
  • 예전에 처음 포트포워딩 시도를 할 때,
  • 이게 포트포워딩이 됐는지 안됐는지 하고도 모를 때가 있다.
  • 그때 외부에서 접속 테스트를 해주는 포트포워딩 체크 웹을 알았더라면…
  • 핸드폰으로 데이터 켜서 접속해보는 귀찮은 짓을 하지 않았겠다..
