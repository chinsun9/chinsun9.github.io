약수의 합

해설

  • 정수를 입력받으면 모든 약수의 합을 반환한다
  • 0, 1인 경우 바로 반환한다
  • 무조건 약수인 1과 자기자신으로 약수가 저장될 배열을 초기화한다
  • 2부터 n의 제곱근까지 반복문을 돌면서 나누어 떨어지는지 판단한다
  • 나누어 떨어지면 약수 배열에 추가한다.
  • 약수는 항상 세트로 존재하기 때문에, 제곱근의 경우가 아니라면 n / index 도 약수로 추가한다.
  • reduce로 약수배열의 합을 구한다.
