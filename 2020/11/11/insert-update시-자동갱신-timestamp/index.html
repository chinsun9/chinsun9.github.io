insert, update시 자동갱신 TIMESTAMP

  • CURRENT_TIMESTAMP은 현재 시간을 의미한다.
  • DEFAULT CURRENT_TIMESTAMP ; insert 시 자동으로 현재 시간이 들어간다
  • ON UPDATE CURRENT_TIMESTAMP ; update 시 자동으로 현재 시간으로 갱신된다.
