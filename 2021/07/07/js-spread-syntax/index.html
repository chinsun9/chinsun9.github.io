js spread syntax `...` 연산자

  • 리액트를 처음 공부할 때 자주 접하면서 알게 된 연산자다
  • 상태를 변경하려면, 완전히 새로운 객체를 대입시켜줘야 한다
  • (참조하는 주소를 이전이랑 다르게 해야 state 갱신이 일어났다고 판단하고 화면을 갱신한다)
  • 자바스크립트에는 괴랄한 문법이 많구나 느꼈었다
  • 한 꺼풀 벗겨내는 연산자라고 생각하면 쉽다
