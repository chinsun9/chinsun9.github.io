geolocation 등 HTTP에서 사용할 수 없는 Web API 목록

  • geolocation 등 특정 Web API들은 localhost 또는 HTTPS 환경에서만 동작한다
  • 그래서 로컬에서 테스트했을 땐 동작하는 것들이 http로 배포했을 때 동작하지 않을 수 있다
