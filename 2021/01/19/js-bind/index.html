js bind

  • typescript 코드다
  • bind 함수를 배워보았다
  • bind를 배우면서 function과 arrow function 차이를 실감하게 되었다
  • arrow function은 this를 바인딩하지 않는다
  • 고로 arrow function은 bind를 사용할 수 없다
  • 이 함수 표현은 메서드 함수가 아닌 곳에 적합하다
